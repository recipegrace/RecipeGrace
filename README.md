### How to run the app?

Source code for the project [www.recipegrace.com](recipegrace.com)
## Build & Run ##

```sh
$ cd RecipeGrace
$ ./sbt
> container:start
> browse
```

If `browse` doesn't launch your browser, manually open [http://localhost:8080/](http://localhost:8080/) in your browser.
